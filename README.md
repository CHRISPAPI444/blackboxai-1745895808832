# CleanCo - Cleaning Services Application

## Project Overview
CleanCo is a web application designed to facilitate cleaning services. This application allows users to hire cleaning services, apply for job openings, and schedule appointments online. Built with a responsive layout, it provides an intuitive user experience, making it easy for clients to request cleaning services or apply to join the CleanCo team.

## Installation
To set up the CleanCo application locally on your machine, follow these steps:

1. Clone this repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd clean-co
   ```

3. Open the `index.html` file in your preferred web browser to view the application.

## Usage
1. **Hire Cleaning Services**: Fill in your name, email, phone number, select the type of service, and provide your address and any additional notes, then submit the form.
2. **Work With Us**: If you are interested in joining the CleanCo team, fill out your details including your resume (optional) and submit your application.
3. **Make an Appointment**: Schedule a cleaning service by providing your details along with the desired appointment date and time.

## Features
- User-friendly forms for hiring cleaning services, job applications, and scheduling appointments.
- Responsive design ensuring accessibility on various devices.
- Modern design with Tailwind CSS for a clean and professional appearance.

## Dependencies
This project uses the following dependencies:
- **Tailwind CSS**: For styling
- **Font Awesome**: For icons
- **Google Fonts**: For custom font (Roboto)

The external libraries are included via CDN in the `index.html` file, so no installation is required.
